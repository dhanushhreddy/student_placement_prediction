# Student Placement & Salary Prediction 🎓

This is a Machine Learning-based web application that predicts the likelihood of a student getting placed in a campus recruitment drive and estimates their potential salary based on various academic and professional parameters.

## 🚀 Features
- **Placement Prediction:** Predicts if a student will be placed (Yes/No).
- **Salary Estimation:** Predicts the expected salary package.
- **Interactive Web UI:** Built using Flask for an easy-to-use experience.

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS
- **Backend:** Flask (Python)
- **Machine Learning:** Scikit-learn, Pandas, Numpy
- **Model Storage:** Pickle

## 📂 Project Structure
- `app.py`: The main Flask application.
- `model.pkl`: The trained ML model for placement.
- `preprocessing.ipynb`: Data cleaning and model training steps.
- `Placement_Prediction_data.csv`: Dataset used for training.
- `templates/`: Contains the HTML files (`index.html`, `home.html`, etc.).

## 🔧 How to Run
1. Clone the repository.
2. Install dependencies: `pip install flask pandas scikit-learn`.
3. Run the app: `python app.py`.
